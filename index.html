About

Helios was the personification of the Sun in Greek mythology.

This project aims to design a modular framework for building custom solar power charge systems that can be monitored remotely.

The base of this project was a conjoined effort by Ed Cheswick (EE), Katie Garrison (ME), Colin Leath (EE), Nathan Sprenkle (CoE) supervised by Dr. Ziang Zhang.
